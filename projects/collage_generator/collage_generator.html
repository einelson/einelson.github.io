Collage Generator

Collage generator is a pseudo random way to create collages. You can specify what you would like for the border, rotation and size. This is a basic creator but was a very fun project. You can add in as many pictures as you would like.

The image to the right was composed of around 28 photos and has a final size of 3000 x 3000 pixels.
